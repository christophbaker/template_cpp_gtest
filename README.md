# C++ Google Test Template 

C++ template for CMake and Google Test with a custom GitHub [Actions](https://github.com/features/actions) and
[Codespaces](https://github.com/features/codespaces).  

```
$❯ cmake -S . -B build
$❯ cmake --build build
$❯ cd build
$❯ make test
$❯ cd ..
```
